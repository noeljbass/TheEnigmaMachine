Enigma Simulator

Configure and run a three-rotor Enigma I

Follow the same basic rules operators had to follow: choose three different rotors, use valid plugboard pairs, record your settings, and reset the same starting positions before decrypting.

Quick guide

  1. Choose three different rotors and a reflector.
  2. Set each rotor's starting position from A to Z.
  3. Add plugboard pairs such as AB CD EF. Each letter can appear once.
  4. Type a message and press Encrypt / Decrypt. Reset the same settings and run the ciphertext again to decrypt.
